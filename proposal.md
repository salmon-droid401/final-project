# Garden Escape

## Repository
[https://github.com/salmon-droid401/final-project.git]

## Description
The player has to jump on platforms and make it to the end of the garden before the giant bird chasing them catches up.

## Features
- Movement
	- The player character moves if the user presses space, 'd', or 'a'.
- Enemy
	- If it touches the player, the level resets. Acts as a time limit for the player to complete the level.

## Challenges
- Creation of platforms that the player can walk and jump on.
- Implement a time limit in a game.

## Outcomes
Ideal Outcome:
- A game that is both challenging and fun. Appealing visuals that show that the player is trying to escape a garden.

Minimal Viable Outcome:
- There is a time limit but the player is not being chased by a giant bird. Visuals show that the player is esccaping a garden, but there is little variety in sprites.

## Milestones

- Week 1
  1. Implement player movement.
  2. Implement platforms. 

- Week 2
  1. Implement time limit for level.
  2. Implement respawn.

- Week 3 (Final)
  1. Implement enemy.
  2. Implement visuals.
